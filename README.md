Women Safety App

Women Safety App is an Android application developed using Kotlin and Firebase to provide quick emergency assistance and improve personal safety through smart mobile technology.

This project was created as a Community Service Project (CSP) with the objective of supporting women’s safety and helping users respond quickly during emergency situations using simple and accessible safety features.

📱 Features

✅ Emergency SOS Alert System
✅ Real-Time Live Location Sharing
✅ Emergency Contact Management
✅ One-Tap Emergency Calling
✅ Firebase Database Integration
✅ Simple & User-Friendly Interface
✅ Fast Emergency Assistance Support

🛠️ Tech Stack
Programming Language: Kotlin
IDE: Android Studio
Backend & Database: Firebase
UI Design: XML
